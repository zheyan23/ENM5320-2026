## About This Course

This course explores the intersection of artificial intelligence and scientific computing, covering topics including:
- Physics-informed machine learning
- Finite difference and finite element methods
- Structure-preserving neural networks
- Hamiltonian and Lagrangian mechanics
- Data-driven modeling with geometric constraints

## Setting Up GitHub Pages

1. Go to your repository Settings
2. Navigate to "Pages" in the left sidebar
3. Under "Source", select the branch (usually `main` or `master`)
4. Select the root folder `/` as the source
5. Click Save

Your site will be published at `https://YOUR-USERNAME.github.io/ENM5320-Spring2026/`

## Local Development

To view the website locally, simply open `index.html` in your web browser.
